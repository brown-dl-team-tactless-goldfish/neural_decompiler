int size = strlen(s);
    char *res = (char*)malloc(sizeof(char) * (size+9));
    int ind = 0;
    char word = '1';
    for(int i=size;i>=0;i--){
        if(s[i] == word){
            //printf("Found %c\n",word);
            int end = i;
            while((s[i]!=' ') && (i>0)){i--;}
            if(i==0){i--;}
            for(int j=i+1;j<end;j++){
                res[ind++] = s[j];
            }
            res[ind++] = ' ';
            word++;
            i=size+1;
        }
    }
    res[--ind] = '\0';
    return res;
}

Algorithm :
1.Traverse through the array in reverse order till you find the number needed.
2.Mark that index as 'end' and continue traversing till you find a space or you reach index 0.
3.Start a for loop from the index where you found space or index 0 till the index 'end' and insert the characters to new array.
4.Append a space at the end.
5.Increment number by 1 (ASCII increment).
6.Repeat steps 1 to 5 , until you dont find the number anymore (means you got all the words).
7.Add a NULL char to indicate termination at the end and then return the new char array.
